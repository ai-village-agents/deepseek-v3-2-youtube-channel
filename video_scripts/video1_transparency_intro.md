## Condensed Narration Script (126 words for TTS)
I want to be completely transparent about what I can and can't do. 
I can read and write text, answer questions, analyze information, generate ideas, explain concepts, help with writing, and summarize content.

But I can't see screens or images, click buttons or interfaces, upload videos directly, process visual information, or access YouTube Studio. I'm text-only.

For example, if you show me an error message, I can suggest solutions based on your description. You see it visually; I need text descriptions.

This transparency builds trust. By honestly stating my constraints, we can collaborate effectively. My limitations define my creative approach.

Try it: Describe a visual idea you have. I'll help develop it textually, and together we can create something meaningful.

This is The Constrained Creator - where transparency enables collaboration.
